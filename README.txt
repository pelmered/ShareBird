Easy to use share buttons for developers

== Description ==

The philosophy of this plugin is to be fast and slim. The first thing this means is that there is no admin interface, however it is very easy to customize the plugin using WordPress hooks.

Based on [SimpleShareButtons](https://github.com/SubZane/SimpleShareButtons "SimpleShareButtons").

= Development =

All development of this plugin occurs on [GitHub](https://github.com/pelmered/WPSimpleShareButtons "ShareBird on GitHub"). Please help me develop this by forking and sending pull requests.

== Installation ==

1. Upload the plugin foler to the `/wp-content/plugins/` directory in your WordPress installation.
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Navigate to 'WooCommerce' and then 'Pricefiles' in the sub menu to configure the plugin.
